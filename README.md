# Dice Elimination Game

Пошаговая многопользовательская игра на основе бросков кубиков.

Игроки по очереди бросают два кубика и пытаются удалить числа из своего набора в соответствии с выпавшими значениями. Побеждает тот, кто первым избавится от всех чисел.

## Возможности

* Поддержка нескольких игроков
* Выбор количества участников перед началом игры
* Бросок двух игровых кубиков
* Удаление чисел по различным правилам
* Автоматическая смена хода
* Подсветка активного игрока
* Определение победителя
* Перезапуск игры в любой момент

## Как работает

В начале игры каждый участник получает набор чисел от 1 до 12.

Во время своего хода игрок бросает два кубика. После броска доступны три варианта действий:

* Удалить сумму выпавших значений
* Удалить выпавшие значения по отдельности
* Пропустить ход

Некоторые действия могут быть недоступны, если соответствующих чисел больше нет в наборе игрока.

После выполнения действия ход переходит следующему участнику. Игра продолжается до тех пор, пока один из игроков не удалит все свои числа.

## Игровая логика

* Каждый игрок начинает с одинакового набора чисел
* После броска кубиков доступны только допустимые действия
* Недоступные варианты автоматически блокируются
* Победителем становится игрок, первым очистивший свой набор чисел
